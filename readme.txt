Figma로 제작한 페이지별 설계도:
https://www.figma.com/file/cTmU198jJafVbjdgTThha1/PowerForecast?node-id=43%3A115


홈페이지를 만든 개요:
    LSTM,CNN(딥러닝)을 이용해서 얻은 전력 예측 데이터를 그래프로 확인하기 위해서 해당 페이지를 제작함
    

페이지 별 설명:
    1.index : 전력 사용을 공장에서 많이 하니깐 바퀴모양 애니메이션을 넣음
    2.lab_explain : 전력 예측이 필요한 이유를 설명하고 전력 예측으로 개발될 사업들을 설명
    3.data_preprocessing : 어디서 전력 데이터와 기온 데이터를 얻었고 데이터 전처리를 어떻게 했는 지 알려줌
    4.deep_learning_model : 딥러닝 모델에서 예측도를 높이기 위해 파라미터 값을 조금 씩 변경한 연구과정을 담음
    5.graph_predict : 영국 런던의 전력 예측 값을 GoogleChart로 두 모델간에 예측도를 비교함
    6.reference : 해당 사이트를 제작하기 위해 참고한 자료 링크를 표시함

개선 해야될 점:
    1.html,css를 단순 반복한 하드코딩: 
    node.js를 사용하면 html에서도 변수,조건문,if문,함수 사용이 가능한데 모든 코드를 직접 다 입력해서
    가독성 떨어지는 코드로 제작함(특히 deep_learning_model 페이지에서  박스 마다 js를 사용해서 상당히 코드가 길어짐)
    --이 페이지를 제작할 때 Node.js를 배우지 못한 상태라서 모든 코드를 하드 코딩으로 제작했음  
    
    2.엉성한 디자인:
    css를 이용해서 디자인을 배치에 맞게 하려고 했으나 해당 페이지에 맞는
    테마 디자인이 잘 안 떠올라서 예쁜 디자인으로 제작하지 못함
    --bootstrap과 같은 프레임워크를 배우고 node.js 연동하는 법을 배워봐야함

    3.데이터 베이스 연동 부재:
    graph_predict에서 데이터 베이스를 웹과 연동하고 바뀐 버전별로
    그래프 값을 출력 했으면 더 많은 데이터를 담을 수 있었을 듯
    -- 주식 같이 실시간으로 변하는 데이터를 Rest-api로  데이터를 DB에  담은 후 Node.js로 해당 데이터를 받아오는 법을 배우면 좋을 듯
